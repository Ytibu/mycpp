# My C++ 项目

这是一个 C++ 学习项目，包含多个模块，用于练习多线程编程、线程池实现、生产者消费者模型等高级 C++ 编程技术。

## 项目结构

- **base/**: 从c++基础语法、c++面向对象、c++11新特性。
- **boost/**: 以面向对象的形式封装c的库函数，包括线程类、网络类等多层封装。
- **Database/**: 使用c\c++连接和使用MySQL和Redis等(不包含sql语句和redis语句)。
- **linux/**: 包含 Linux 系统编程相关的文件、进程、线程、网络。

## 主要功能

- **多线程编程**: 使用 C++ 标准库和 POSIX 线程 API 实现多线程程序。
- **线程池**: 提供两种线程池实现，一种是面向对象设计，另一种是基于回调函数的设计。
- **生产者消费者模型**: 使用条件变量和互斥锁实现线程安全的任务队列。
